# Java Streams SEO POC

A small Next.js App Router project that demonstrates basic SEO implementation for a Java Streams tutorial page.

## Overview

This project includes:

- A homepage at `/`
- A blog article page at `/blog/java-streams`
- Page-level SEO metadata
- Open Graph metadata for social sharing
- A generated sitemap at `/sitemap.xml`
- A generated robots file at `/robots.txt`
- An optimized tutorial image using `next/image`

## Routes

| Route | Purpose |
| --- | --- |
| `/` | Homepage introducing the Java Streams tutorial |
| `/blog/java-streams` | Full Java Streams tutorial article |
| `/sitemap.xml` | Search engine sitemap generated by Next.js |
| `/robots.txt` | Robots rules generated by Next.js |

## Key Files

| File | Description |
| --- | --- |
| `app/page.tsx` | Homepage content |
| `app/blog/java-streams/page.tsx` | Java Streams blog article and SEO metadata |
| `app/layout.tsx` | Root layout and default metadata |
| `app/sitemap.ts` | Sitemap configuration |
| `app/robots.ts` | Robots configuration |
| `public/java-streams.png` | Open Graph and article image |
| `app/globals.css` | Global styles and article typography |

## SEO Features

The blog page defines:

- SEO title
- Meta description
- Open Graph title
- Open Graph description
- Open Graph image
- Twitter card metadata generated by Next.js

The project also exposes:

```text
http://localhost:3000/sitemap.xml
http://localhost:3000/robots.txt
```

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open:

```text
http://localhost:3000
```

## Verify

Run lint:

```bash
npm run lint
```

Run a production build:

```bash
npm run build
```

Check the SEO routes:

```bash
curl http://localhost:3000/sitemap.xml
curl http://localhost:3000/robots.txt
```

## Domain Setup

The current sitemap, robots, and metadata base use:

```text
https://your-domain.com
```

Before deploying, replace this placeholder with the real production domain in:

- `app/layout.tsx`
- `app/sitemap.ts`
- `app/robots.ts`
